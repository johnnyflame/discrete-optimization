# A roadmap of algorithmic topics

## Why every software developer study algorithms?

- A great way to add value to your software product or service.
- Improve your day to day work on the job
- Improve your position in the job market

## Beginner tier

- Concepts
  - Divide and conquer
  - Recursion
  - Dynamic programming
- Search
  - Binary search
  - DFS
  - BFS
- Recursion
  - Tree traversal
    - Pre, post, level order
- Sorting algorithms
  - Basic N^2 sorts:
    - Insertion sort
    - Selection sort
  - N log N sorts:
    - Quicksort
    - Mergesort
    - Heapsort

## Intermediate tier

- Search
  - Backtracking
  - Branch and bound
  - Constraint programming
- Sort
  - Non-comparison sorts
    - Radix sort
    - Counting sort
    - Bucket sort
- Dynamic programming

## Advanced topics

Often these are for solving NP-complete problems.

- String processing
  - KMP
  - Boyce Moore
  - Manacher's algorithm
- Optimization
  - Dynamic programming
  - Branch and bound
